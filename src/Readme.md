Library Management System

This project is a Java Swing based desktop application that manages books and members in a library.

Features:

Add and delete books
Add and delete members
Borrow and return books
Search books
View borrowed books
Persistent storage using text files
Admin login system

Technologies:

Java
Swing GUI
File Handling

How to run:

Open project in VS Code
Run Main.java
